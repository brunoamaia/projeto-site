Brunoflix

Projeto desenvoldido durante a Imersao React . Que foi promovida pela Alura entre os dias 27 e 31 de julho de 2020.

O Brunoflix é uma aplicação com o intuito de copiar (recriar) a interface da famosa Netflix. O principal intuito foi aprender mais sobre a utilização do React na criação de páginas web. Foram também apresentadas algumas maneiras de aprimorar o layout. Tornando a experiência do usuário mais agradável.

O banner a seguir, mostra um pouco da interface que foi criada. É importante ressaltar, que o conteúdo ainda será modificado. O foco foi a interface, portanto, primeiramente serão terminadas as funcionalidades. Posteriormente, o conteúdo será editado e algumas funcionalidades extras implementadas.

O projeto foi postado nas plataformas da Vercel e Heroku. Para acessá-lo e ver essas funcionalidades, clique aqui.

Caso queira olhar o código e quem sabe contribuir, segue o Link para o Repositório no GitHub.
