Ecoleta

Projeto desenvoldido durante a Primeira Next Level Week. Que foi promovida pela Rocketseat entre os dias 01 e 07 de junho de 2020.

O Ecoleta é uma aplicação voltada à coleta de materiais recicláveis e lixo orgânico. A aplicação possui tanto versão web quanto a versão mobile (Aplicativo). Para o desenvolvimento de toda a estrutura do Ecoleta, foram utilizadas as seguintes ferramentas:

  • Node.js: para a criação do servidor (Backend) e suas funcionalidades utilizando JavaScript;
  • React: para desenvolver uma plataforma web dinâmica (Frontend);
  • React Native: para fazer um aplicatico que, a partir do mesmo código, possa ser gerada tanto a versão para Android, quanto para IOS

O vídeo a seguir apresenta o funcionamento do Ecoleta. É importante ressaltar que devido ao fato de ser desenvoldido em apenas uma semana, algumas funcionalidades estão na versão web e outras na versão Mobile.
